
# Machine Learning for Options

A concise collection of notebooks demonstrating machine learning methods for option pricing, implied volatility estimation, fast numerical pricing, and hedging.

## Contents

* **01_black_scholes_ml_pricing.ipynb** — Compares Black–Scholes pricing with ML-based approximators.
* **02_ml_implied_vol.ipynb** — Predicts implied volatility using supervised learning.
* **03_fast_option_pricing.ipynb** — Examines high-speed pricing using surrogate models.
* **04_hedging.ipynb** — Tests ML-driven hedging strategies and error reduction.

## Usage

Clone the repository and run the notebooks with Jupyter:

```bash
git clone https://github.com/<your-username>/ml_options.git
cd ml_options
jupyter notebook
```

## Requirements

Python 3.9+, NumPy, Pandas, Matplotlib, Scikit-learn, and optionally TensorFlow or PyTorch.
